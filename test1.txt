1. Git 설치 
   → Git Bash 실행후 'git + Enter' : 깃 명령에서 사용할 수 있는 여러 옵션이 표시된다면 제대로 설치된 것.
2. Gti 환경설정
   → 깃을 사용하기 전에 먼저 사용자 정보를 입력해야 한다.
       '버전'을 저장할떄마다 그 버전을 만든 사용자 정보도 함께 저장되기 때문
      이를통해 어떤 '버전'을 누가, 언제, 어떤 메세지를 남겨 만들었는지 쉽게 파악할 수 있다.
      Git Bash 실행후 
      git config --global user.name "이름or별명"
      git config --global user.email "깃허브 에서 사용하는 email주소"
      
      * --global 옵션을 추가하면 현재 컴퓨터에 있는 모든 저장소에서 같은 사용자 정보를 사용하도록 설정한다는 의미
        여기에서는 '깃 허브' 계정을 하나만 사용한다고 가정하고 --global 옵션을 사용함
        깃 허브 계정이 여러개라면 각 계정마다 깃 환경을 따로 설정해야 함
     
      * config: configuration 배열, 배치, [배열,배치]형태 / 환경설정
